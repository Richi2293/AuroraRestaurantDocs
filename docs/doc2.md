This is the structure of the React Native project:

```
AuroraRestaurant/ 
├── \__tests__ 
├── android 
├── ios 
├── node_modules 
├── src 
    └── api (fake api data)
    └── assets (assets files)
    └── components (components)
    └── screens (screens)
    └── translations (file of translations )
    └── app.tsx (entry file for app)
    └── settings.tsx (settings, colors, font sizes, ecc...)
    └── themeStyles.tsx (global file style)
├── .buckconfig 
├── .eslintrc.js 
├── .flowconfig 
├── .gitattributes 
├── .gitignore 
├── .prettierrc.js 
├── .watchmanconfig 
├── app.json 
├── babel.config.js 
├── index.js 
├── jest.config.js 
├── metro.config.js 
├── package.json 
├── react-native.config.js 
├── README.md 
├── tsconfig.json 
└── yarn.lock 
```
